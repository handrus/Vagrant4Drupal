Vagrant4Drupal

A Puppet powered Vagrant box for Drupal projects.

Download as .zip Download as .tar.gz View on GitHub

A great and fast way to bootstrap your Drupal project.

Vagrant4Drupal is a packaged virtual environment to run a Drupal project out of the box. It uses VirtualBox, Puppet and Vagrant to provide, automagically, a LAMP stack with all Drupal needs already installed.

There's no need to install LAMP stuffs, Git, Drush anymore. Just checkout Vagrant4Drupal, run vagrant up and see the magic happen.

$ git clone https://github.com/joelwallis/Vagrant4Drupal.git
$ cd Vagrant4Drupal
$ vagrant up

Vagrant4Drupal is a young and simple project, and you're encouraged to fork it and modify for your needs. But if you created something cooler or added something common to any Drupal project that wasn't provisioned, share back with the project and help it to grow. :)
